
# Gatsby Theme Starter

Boilerplate for developing Gatsby themes

## Getting Started

1. Clone this repo
2. Remove git history with `rm -rf .git`
3. Change the name in:
  - `package.json`
  - `gatsby-config.js`
  - `demo/package.json`
  - `demo/gatsby-config.js`
4. Edit the source code to create a theme

## Running the demo

```sh
cd demo
npm install
npm start
```
